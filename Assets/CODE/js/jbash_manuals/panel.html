Manual for PANEL [closeall] [panel_name]

panel holds some commands concerning the GML-panels.

Parameters:
closeall: Closes all panels. Can be alone or before or after the panel_name.
panel_name: Open the panel with the name panel_name.

Examples:
panel closeall
panel my_panel
panel closeall my_panel
panel my_panel closeall
