Manual for SHOW what [id_or_name]

show debug info about the given stuff.

Use items, rooms, panels or sounds for what

Parameter:
what: Show debug info about all the stuff given as parameter.
id_or_name: ID or name of a specific item in the list.

You can show debug info about: items, sounds, panels and rooms

It will then show a list with all the given items and a number for them.
Use "show item x" or "show panel panelname" or such for info about a specific item.
Example:
show items
show sound 0
show room my_room
