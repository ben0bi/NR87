Manual for ITEM [unfocus] or [item_name focus [x y]]

item holds some commands concerning the GML-items.

Parameters:
unfocus: Must be only parameter. Hides all mouseover-images of the items in the room.
item_name focus: show the mouseover-image of the item named item_name. Jumps to the room where the item is in, if it is in another room
x y: Move the room to the given position. Can only be used with focus right now.

Examples:
item unfocus
item my_item focus
item my_item focus 30 -150
